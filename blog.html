1. What is DOM and what is the purpose of DOM?

Ans: DOM means Document Object Model. It represents the page so that programs can change the document structure, style, and content.

2. How will you select HTML elements using DOM? Name the DOM methods.

Ans: There are 5 ways we can select elements in a DOM using selectors.

  • getElementsByTagName()
  • getElementsByClassName()
  • getElementById()
  • querySelector()
  • querySelectorAll()

3. What is event bubble?

Ans: Event bubbling is a type of DOM event propagation where the event first triggers on the innermost target element, and then successively triggers on the ancestors of the target element in the same nesting hierarchy till it reaches the outermost DOM element or document object

4. What is a callback function and why will you use it?

Ans: Callback function is a function which pass as parameters to other functions and call them inside the outer functions.
Callback function make sure that a function is not going to run before a task is completed but will run right after the task has completed. It helps us develop asynchronous JavaScript code and keeps us safe from problems and errors.
